CLI Commands:
# Sync all feeds
flask telegram sync

# Sync specific feed
flask telegram sync-feed 1

# Add new feed
flask telegram add-feed @channel "Feed Name"

# Test connection
flask telegram test-connection

# List all feeds
flask telegram list-feeds
Setup Instructions:
  1. Get Telegram API credentials from my.telegram.org/apps
  2. Add TELEGRAM_API_ID and TELEGRAM_API_HASH to your .env file
  3. Test connection using the button above
  4. Add Telegram channels using the CLI or admin interface
  5. Bot will automatically sync every 30 minutes
